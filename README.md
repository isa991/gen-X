# gen-X
Aplicação web médica de avaliação da síndrome do X frágil; Calcula o score do paciente a partir dos sintomas para realizar a tomada de decisão em relação ao exame diagnóstico. 

Trabalho apresentado para a disciplina de Experiencia Criativa: Criando Soluções Computacionais, do curso de Bacharelado em Ciências da Computação, da Pontificia Universidade Católica do Paraná, campus Curitiba.
